# tweeder

> Displays metrics about npm modules.

## Install

> npm install --save tweeder

## Usage

```js

const tweeder = require('tweeder');

packStat('pakalupapito').then(tweets => {
	console.log(tweets);
	/*
	{
		COMMING SOON
	}
	*/
});
```
